# Ola-Data-Analysis-Dashboard
OLA Data Analysis Dashboard is a Power BI project that analyzes ride-booking data to uncover insights into bookings, revenue, cancellations, vehicle performance, and customer behavior. Built using Power BI, DAX, and Power Query, the dashboard enables data-driven decision-making through interactive visualizations and KPI tracking.

🚖 OLA Ride Booking Data Analysis Dashboard
1. Project Title & Headline

OLA Ride Booking Data Analysis Dashboard | Power BI & SQL

An end-to-end data analytics project that transforms OLA ride-booking data into actionable business insights using SQL for data analysis and Power BI for interactive dashboard visualization.

2. Short Description & Purpose

This project analyzes OLA ride-booking data to understand customer behavior, ride performance, cancellation trends, payment preferences, and revenue generation. Using SQL queries and Power BI visualizations, the dashboard helps stakeholders monitor key business metrics and identify opportunities to improve operational efficiency and customer experience.

The goal is to convert raw ride data into meaningful insights that support data-driven decision-making.

3. Tech Stack
Power BI – Dashboard development and data visualization
SQL (MySQL) – Data extraction, transformation, and analysis
Power Query – Data cleaning and preparation
DAX – KPI calculations and measures
Excel/CSV Dataset – Source data storage and management
4. Data Source

The dataset consists of OLA ride-booking records containing information such as:

Booking ID
Customer ID
Vehicle Type
Ride Distance
Booking Status
Customer Rating
Driver Rating
Payment Method
Cancellation Reasons
Booking Value
Incomplete Ride Details

The dataset was analyzed using SQL and visualized through Power BI dashboards.

5. Features / Highlights
📈 Booking Performance Analysis

Track successful, cancelled, and incomplete rides.

Example: Identify the total number of successful bookings and cancellation rates.

🚗 Vehicle Type Analysis

Analyze ride distance and performance across vehicle categories.

Example: Calculate average ride distance for Prime Sedan, Mini, Auto, and other vehicle types.

❌ Cancellation Insights

Monitor customer and driver cancellation patterns.

Example: Find rides cancelled by drivers due to personal or vehicle-related issues.

⭐ Rating Analysis

Evaluate customer satisfaction and driver performance.

Example: Compare average customer ratings across vehicle types and identify highest-rated services.

💳 Payment Method Analysis

Understand customer payment preferences.

Example: Retrieve all rides paid through UPI and compare payment mode usage.

💰 Revenue Analysis

Measure booking value and business performance.

Example: Calculate total revenue generated from successfully completed rides.

🚨 Incomplete Ride Monitoring

Track rides that were not completed and analyze their reasons.

Example: Identify operational issues causing incomplete rides.

The dashboard provides a comprehensive view of OLA's operational performance, enabling stakeholders to track KPIs, identify trends, improve customer satisfaction, reduce cancellations, and make informed business decisions through data-driven insights.
